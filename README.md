# 🎮 Simon Says Game

A classic memory game built with pure **HTML**, **CSS**, and **JavaScript**. Watch the sequence, repeat it back, and see how far you can go!

---

## ✨ Features

- ⌨️ Press any key to start the game
- 🎨 Four colored buttons — Red, Yellow, Green, Purple
- 🔢 Sequence grows by one each level
- 💡 Buttons flash to show the pattern
- ❌ Game Over screen with final score on wrong input
- 🔁 Auto-reset after game over — press any key to restart
- 💡 Pure Vanilla JS — no libraries or frameworks

## ⚙️ How It Works

1. Press any key to start the game
2. A random colored button flashes — remember it!
3. Click the buttons in the same order
4. Each level adds one more step to the sequence
5. Click the wrong button → Game Over, your score is shown
6. Press any key again to restart
